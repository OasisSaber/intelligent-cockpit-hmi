# 文档索引

先按任务读取下表中的最小文件集合；不要扫描整个 `docs/`。

| 任务 | 默认入口 | 读取范围 |
| --- | --- | --- |
| 项目概览 | `architecture.md` | 仅需系统边界时 |
| 开发、验证、版本协作 | `development.md`；叶子任务委派另见 `ai-collaboration/OPENCODE_FIRST_WORKFLOW.md` | 日常开发时 |
| 当前范围与验收 | `ai-collaboration/PROJECT_DECISION_BASELINE.md` | 需求或架构改变前 |
| 当前工作状态 | `ai-collaboration/CURRENT_HANDOFF.md` | 每次开始时 |
| 设计到代码 | `design/`、`contracts/` | 仅相关屏幕/协议任务 |
| 毕设论证与材料映射 | `00-10-*.md`、`school/` | 论文、答辩或学校要求时 |
| 历史资料 | `archive/`、`../PreDesign/idea-archive/` | 仅追溯设计过程时 |

## 当前协作资料

`ai-collaboration/` 保存 handoff、开发标准、已冻结决策和任务说明。不要整目录加载；按 `ai-collaboration/README.md` 的职责表选择文件。

## 低频资料

`archive/`、`research/`、`school/`、编号 `00-10` 文档和 `design/` 均保留为毕业设计证据或专项说明。它们不是日常实现任务的默认上下文。
