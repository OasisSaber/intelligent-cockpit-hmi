# 文档索引

先按任务读取下表中的最小文件集合；不要扫描整个 `docs/`。

| 任务 | 默认入口 | 读取范围 |
| --- | --- | --- |
| 项目概览 | `architecture.md` | 仅需系统边界时 |
| 开发与验证 | `development.md`、`DEVELOPMENT_STANDARDS.md` | 日常开发时 |
| 当前范围与验收 | `project/DECISION_BASELINE.md` | 需求或架构改变前 |
| 当前产品进度 | `project/PROJECT_PROGRESS.md` | 排期、验收或风险判断时 |
| 实施路线 | `project/IMPLEMENTATION_ROADMAP.md` | 规划后续产品工作时 |
| 设计到代码 | `design/`、`contracts/` | 仅相关屏幕/协议任务 |
| 毕设论证与材料映射 | `00-10-*.md`、`school/` | 论文、答辩或学校要求时 |
| 历史资料 | `archive/`、`../PreDesign/idea-archive/` | 仅追溯设计过程时 |

## 低频资料

`archive/`、`research/`、`school/`、编号 `00-10` 文档和 `design/` 均保留为毕业设计证据或专项说明。它们不是日常实现任务的默认上下文。
