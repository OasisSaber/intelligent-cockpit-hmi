## 8. 冻结门槛

### 风格门槛

- [ ] 四屏各有代表性 Day/Night 主画面
- [ ] 字体、颜色、材质、间距、信息层级和动效语言一致
- [ ] HUD/Cluster 的目标距离可读性无阻断问题
- [ ] Center/Passenger 触控尺寸满足基线
- [ ] 字体、图标、图片、三维和音频资产可交付并有许可证据

### 流程门槛

- [ ] 导航任务接力可完整演示
- [ ] 关键告警升级与恢复可完整演示
- [ ] 副驾媒体与隐私可完整演示
- [ ] 关键正常、告警、降级和恢复状态均有证据
- [ ] 形成性测试的阻断问题已经解决或接受风险

### 技术可实现性

- [ ] 已确认可映射到现有 React + FastAPI 架构
- [ ] 没有违反四屏权限或共享状态单一来源
- [ ] 没有超出性能预算的已知阻断项
- [ ] 没有未解决 P0
